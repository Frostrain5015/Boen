# 义务教育语文课程标准（2022年版）· 核心摘要

## 四大语文实践活动（学习领域）

### 识字与写字
- 汉语拼音：声母、韵母、整体认读音节、拼读规则
- 汉字识记：常用汉字3500个（小学2500 + 初中1000）
- 书写规范：笔画、笔顺、间架结构、硬笔楷书

### 阅读与鉴赏
- 文学阅读：小说、散文、诗歌、戏剧的阅读理解与审美鉴赏
- 实用类阅读：说明文、议论文、新闻、非连续性文本
- 古诗文阅读：文言文基础（实词虚词、句式）、古诗词鉴赏
- 整本书阅读：名著阅读方法与思维训练

### 表达与交流
- 口语交际：倾听、表达、应对能力，即兴发言与讨论
- 写作：记叙文、说明文、议论文、应用文的写作能力
- 书面表达：结构清晰、语言准确、逻辑严密

### 梳理与探究
- 综合性学习：跨学科主题探究、信息搜集与整理
- 语言规律探究：语法知识、修辞手法、文体特征

## 核心素养四维目标

1. **语言运用**：正确、规范、有效地理解和运用国家通用语言文字
2. **思维能力**：发展直觉思维、形象思维、逻辑思维、辩证思维和创造思维
3. **审美创造**：感受、理解、欣赏、评价语言文字之美，具有初步的审美创造力
4. **文化自信**：认同中华文化，传承中华优秀传统文化

## 各学段要点

### 第四学段（7-9年级）
- 阅读：能熟练运用默读、浏览、略读等阅读方法；阅读文学作品能有自己的情感体验
- 写作：45分钟完成不少于500字的习作；学写记叙文、简单的说明文和议论文
- 古诗文：背诵优秀诗文80篇（段）；阅读浅易文言文，能借助注释理解基本内容
- 名著：每学年阅读两三部名著，掌握阅读整本书的方法
